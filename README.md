# ChatGPT Voice UI

This project is a voice UI implementation featuring an OpenAI-style shader.

**Original Repository:** [openai/openai-realtime-agents](https://github.com/openai/openai-realtime-agents.git)

## Project Status
- **Current State:** This project is a bit buggy and non-performant at the moment. It is under active development.
- **Running Costs:** Please be aware that the running cost for testing and development is approximately **$2-4 USD per day** depending on usage.

## Setup
- This is a Next.js TypeScript app. Install dependencies with `npm i` or `pnpm i`.
- Add your `OPENAI_API_KEY` to your environment variables (e.g., in a `.env` file).
- Start the development server with `npm run dev` or `pnpm dev`.
- Open [http://localhost:3000](http://localhost:3000) in your browser.
